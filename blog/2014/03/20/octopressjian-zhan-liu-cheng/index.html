Octopress建站流程

Mar 20th, 2014

一.设置Octopress

  1. 检查Ruby环境

    确保Ruby环境为1.9.3,若不是，则下载RVM,安装1.9.3版本

     curl -L https://get.rvm.io | bash -s stable --ruby
 rvm install 1.9.3
 rvm use 1.9.3
 rvm rubygems latest

  2. clone Octopress

     git clone git://github.com/imathis/octopress.git octopress
 cd octopress

  3. 安装依赖(dependencies)

     sudo gem install bundler
 sudo bundle install

  4. 安装默认主题

     rake install

二.部署到Github Pages

  1. 若本机无SSH Key,则生成Github SSH Key

    1. 生成SSH key

       $ ssh-keygen -t rsa -C "your_email@example.com"
 $ ssh-add id_rsa

    2. 添加SSH Key到GitHub

       $ pbcopy < ~/.ssh/id_rsa.pub #拷贝到clipboard

      在github的Account Settings添加SSH Key

    3. 测试环境输出

       $ ssh -T git@github.com

  2. 创建Github Repository

    • 名字格式为username.github.io,username需要是github的用户名。即xuguoxing.github.io

    • Initialize this repository with a README 项不勾选

  3. 设置 github pages

     rake setup_github_pages

    该命令会询问repo地址，并在_deploy目录下设置master branch

     rake generate
 rake deploy

    生成blog,拷贝生成的文件到_deploy/,添加到git,commit并push到master branch. 等待大约10分钟

  4. source上传到Github

     git add .
 git commit -m 'your message'
 git push origin source
